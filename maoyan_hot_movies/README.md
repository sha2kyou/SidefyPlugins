## 猫眼热映电影插件

自动获取猫眼电影当前影院热映信息，每天在日历中展示 TOP5 热门电影，让你随时了解院线最新动态。

### 功能特性

- **实时热映**：每天自动获取当前影院最热门的 5 部电影
- **智能更新**：每日更新一次，确保信息新鲜及时
- **一键直达**：点击事件直接跳转到猫眼电影网站查看详情
- **简洁展示**：清爽的电影列表，专注核心信息
- **全天显示**：作为全天日程，便于随时查看

### 显示内容

插件会在日历中创建一个全天事件，包含：

- **标题**：影院热映电影 TOP5
- **内容**：按排名顺序显示当前最热门的 5 部电影
  - 1. 电影名称1
  - 2. 电影名称2
  - 3. 电影名称3
  - 4. 电影名称4
  - 5. 电影名称5
  - 点击查看更多电影信息
- **颜色**：橙色主题，突出影院氛围

### 使用场景

- **观影决策**：快速了解当前院线热门，帮助选择观影内容
- **院线追踪**：随时掌握电影院最新上映动态
- **周末规划**：为周末观影计划提供参考
- **电影爱好者**：及时了解电影市场热点趋势

### 使用方法

1. 下载安装插件即可开始使用
2. 每天会自动在日历中显示当日热映电影
3. 点击事件可跳转到猫眼电影网站查看详细信息
4. 第二天自动更新为最新热映信息

### 更新频率

- **每日更新**：每天自动获取最新热映电影信息
- **当日缓存**：一天内多次查看显示相同内容，避免重复加载
- **自动刷新**：午夜自动更新为最新数据
